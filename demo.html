Animations are supported too!

Classes can be added and removed using data-animate.
Eg.: data-animate="A +B -C" adds .A, .B and removes .C.

All animations from Animate.css are bundled!

Automatically triggered on fragment descendants!

Speed can be controlled with .infinite or data-duration.

Animations can be delayed with data-delay!

Fragments order

Control fragment order by using data-index, a variation around data-fragment-index that takes existing unindexed fragments into account.
